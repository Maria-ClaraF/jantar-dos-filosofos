# Jantar dos Filósofos - Sincronização de Threads

Projeto desenvolvido para a disciplina de Sistemas Operacionais da **UFU**. O objetivo é solucionar o problema clássico do Jantar dos Filósofos, demonstrando a ocorrência de *deadlock* e a implementação de uma solução utilizando a técnica do filósofo canhoto (inversão de hierarquia de recursos).

## 🚀 Como executar

### Pré-requisitos
* Compilador GCC
* Biblioteca pthread

### Compilação
Basta utilizar o Makefile incluso:
```bash
make
